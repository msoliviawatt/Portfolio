Web Server Setup With Raspberry Pi 5
Home About Projects Essays Contact Tutorials

Make sure you have downloaded the Raspberry Pi OS from here

Flash it to a micro SD card and set up your system (Check the documentation if you're unfamiliar)

1. Open a terminal on your Raspberry Pi, or ssh into it from a separate computer

2. Run the following commands in the order shown

sudo apt update
sudo apt upgrade -y
sudo apt full-upgrade

3. Install the LAMP stack (Linux, Apache, MariaDB, PHP)

sudo apt install apache2 mariadb-server php php-mysql libapache2-mod-php php-cli php-curl php-gd php-xml php-mbstring unzip
sudo systemctl enable apache2
sudo systemctl start apache2
sudo mysql_secure_installation
sudo mysql -u root -p

4. Create a database and user for your web server. Replace "your_database_name" with something like wordpress, and the other replacements are self-explanatory

CREATE DATABASE your_database_name;
CREATE USER 'your_username'@'localhost' IDENTIFIED BY 'your_password';
GRANT ALL PRIVILEGES ON your_database_name.* TO 'your_username'@'localhost';
FLUSH PRIVILEGES;
EXIT;

5. Run the following

cd /var/www/html
sudo rm index.html
sudo wget https://wordpress.org/latest.tar.gz
sudo tar -xvzf latest.tar.gz
sudo mv wordpress/* .
sudo rm -rf wordpress latest.tar.gz
sudo chown -R www-data:www-data /var/www/html
sudo chmod -R 755 /var/www/html
sudo cp wp-config-sample.php wp-config.php
sudo nano wp-config.php

6. Edit the database name, username, and password in the wp-config.php file

define( 'DB_NAME', 'wordpress' );
define( 'DB_USER', 'your_username' );
define( 'DB_PASSWORD', 'your_password' );
define( 'DB_HOST', 'localhost' );

7. Install phpMyAdmin

sudo apt install phpmyadmin

8. Select Apache2 when prompted

9. Select Yes when prompted to use dbconfig-common to set up the database

10. Enter the password you set for your database user

12. Enter the password you set for your database user again

13. Restart Apache2

sudo systemctl restart apache2

14. Open your web browser and go to http://localhost/phpmyadmin

15. Log in with the username and password you set for your database user

16. You can now create and manage databases for your web server
