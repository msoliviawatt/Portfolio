How to Enable SSH on Raspberry Pi
Home About Projects Essays Contact Tutorials

Make sure you have downloaded the Raspberry Pi OS from here

Flash it to a micro SD card and set up your system (Check the documentation if you're unfamiliar)

1. Open a terminal on your Raspberry Pi

2. Run the following command.

sudo raspi-config

3. Select "interface options".

4. Enable SSH

Alternatively...

1. Open the start menu from your Raspberry Pi home screen interface

2. Select "Preferences".

3. Select "Raspberry Pi Configuation".

4. Select the "Interfaces" tab.

5. Select "Enable" next to SSH.
